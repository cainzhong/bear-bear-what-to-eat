# Bear Bear Lunch Frontend

This Angular app shows a single page with a button "熊熊今天吃什么" and displays the lunch recommendation provided by the backend API.

## Prerequisites

- Node.js 18 LTS or newer
- npm (bundled with Node.js)

## Getting Started

```powershell
cd frontend
npm install
npm start
```

The dev server runs on http://localhost:4200 and proxies requests directly to the backend running on http://localhost:8000.
